JSFX Scripts

Welcome to my DSP effect scripts for ReaJS (or REAPER), a free and powerful VST plugin that compiles and runs script code on-the-fly and executes it as an integrated DSP effect. The scripts are written in a language called JSFX (Jesusonic FX), developed by Cockos in-house. If optimized properly, it can be quite fast. While not as fast as native code (say, written in C), it is simple to write and for simple effects it doesn't really matter, so no compiled code for now.

You won't find conventional processing effects like EQ, compressors, dynamics, saturators, reverbs, etc here. There's plenty of excellent of those to go around, even free, so I didn't find much point in wasting time coding yet another EQ or the like. Instead, you will find completely weird or shall I say innovative processors that I personally had a need of. In fact, the reason I even wrote them was because I couldn't find anything similar in the plugin universe, free or not, so I got really annoyed at one point. The exception are some simple MIDI tools I made just because I had a need for a lightweight MIDI handler that I couldn't find in any other place — those are not special or crazy or anything.

Feel free to browse around the categories for my scripts and download them, hopefully you find some use for them.
