Channel Randomizer

This simple JSFX script randomizes the channel of incoming MIDI events. It keeps track of what channel it randomized for Note On events so that the same channel is used for a Note Off event for the same note, but otherwise it completely randomizes the channel. Yes it will randomize the channel of any other events other than notes as well, with no relationship between one another. If you don't want that, have the other events filtered out and join them later after this script (so they will be passed through), but that depends on your DAW routing.

Download it here

There are only two parameters:

  • First Channel: Specifies the first channel that can be output; this is included as a possible channel.
  • Last Channel: Specifies the last channel that can be output; this is also included as a possible channel.

You could use this script to randomize the triggering of specific sounds or synths, for example. Just route each channel to a different place and then you'll have your notes randomly sent to a different one.
