Place the "MIDI Tools" directory under your JSFX "Effects" directory. The "Tools" directory is not JSFX, it holds the AutoHotkey helper scripts that create the Key Tuner scales for ReaJS from the text file format. So, you can place it anywhere you find it most accessible to run when needing to convert, just double click it after associating .ahk with AutoHotkey. :)

You can remove the rest of the files and the archive to keep your drive clean, if you want.

If you have no idea what this is, see: https://sites.google.com/site/octarone/code/midi-tools/key-tuner
